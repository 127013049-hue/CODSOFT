# Credit Card Fraud Detection 💳

## 📌 Project Overview
This project detects fraudulent credit card transactions using machine learning techniques.

---

## 🎯 Objective
To classify transactions as legitimate or fraudulent based on transaction data.

---

## 📊 Dataset
- Credit Card Fraud Dataset: https://www.kaggle.com/mlg-ulb/creditcardfraud

---

## 🛠️ Techniques Used
- Data Preprocessing
- Handling Class Imbalance
- Classification Models
- Model Evaluation (Accuracy, Precision, Recall)

---

## 🧠 Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 📈 Outcome
A trained classification model capable of detecting fraudulent transactions effectively.
