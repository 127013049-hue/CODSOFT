# Sales Prediction 📊

## 📌 Project Overview
This project predicts product sales based on advertising expenditure using machine learning.

---

## 🎯 Objective
To analyze advertising data and build a regression model to predict future sales.

---

## 📊 Dataset
- Advertising Dataset: https://www.kaggle.com/datasets/bumba5341/advertisingcsv

---

## 🛠️ Techniques Used
- Linear Regression
- Data Visualization
- Feature Analysis
- Model Evaluation

---

## 🧠 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📈 Outcome
A regression model that predicts sales based on advertising spending.
